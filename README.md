# Crossfade

Crossfade is a platform built to make it easier for friends to connect through music, even if they use different streaming services. It focuses on letting people share playlists, see what others are listening to, and listen together with streaming rooms, without being locked into a single application.

---

# Why this exists

This project started because my friend and I ran into the problem that we both use different music streaming platforms, but we still want to share music and listen together easily.

Crossfade is my attempt to solve that by:

* Syncing playlists across different services
* Making shared streaming possible across platforms
* Bringing listening activity and user stats into one place

The goal is to make it easier to share and experience music together, wherever it comes from — and to give my friends one less reason to tell me to switch to Spotify every time we talk about music.

---

## Project status

This is an early-stage project. The current focus is on building a solid backend foundation and getting core services working through Docker images.
